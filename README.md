# NodeJS App

## [Web Viewer(click)](https://nodejsappsens.herokuapp.com/)


+ init
    ```git
    git init
    npm init -y
    ```
+ install express 
    ```
    npm install --save express
    ```
+ install ejs / ejs-lint
    ```
    npm install --save ejs

    npm install -g ejs-lint
    ```

    ```
    ─node_modules

    │  ├─ejs
    │  │  ├─bin
    │  │  └─lib

    │  ├─express
    │  │  └─lib
    │  │      ├─middleware
    │  │      └─router

    └─public

    └─views
        ├─components
        ├─layouts
        └─pages
    ```
